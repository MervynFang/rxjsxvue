# rxjsxvue
### dependencies
- rxjs 5.0.x
- vue 2.0.x
- vue-rx

### Start

```
npm install
```
Install dependency

```
npm run dev
```
Development mode `open http://localhost:3600 in Browser`

```
npm run build
```
Build for production

```
npm run lint
```
Run ealint

```
npm run dist
```
Production mode `open http://localhost:3601 in Browser`
