# rxjsxvue
### Dependencies
- rxjs 5.0.x
- vue 2.0.x
- vue-rx

### Start

```
npm install
// or
yarn / yarn install
```
Install dependency

```
npm / yarn run dev
```
Development mode `open http://localhost:3600 in Browser`

```
npm / yarn run build
```
Build for production

```
npm / yarn run lint
```
Run eslint

```
npm / yarn run dist
```
Production mode `open http://localhost:3601 in Browser`
